## Как работают команды

1. `ls` - просмотр содержимого(с флагом -l)
2. `cd` - переход в папку
3. `cat` - вывод содержимого файла
4. `cp` - копирование файлов и папок(флаг -r)
5. `mv` - перемещение файлов в папки и переименование файлов
6. `rm` - удаление файлов и папок(флаг -r); невозможно удалить корневой путь
7. `zip`/`unzip` - работа с ZIP архивами
8. `tar`/`untar` - работа с TAR архивам
9.`grep` - поиск по содержимому файлов
10. `undo` - отмена команд mv, rm, cp


## Обработка ошибок
Ошибки не останавливают программу и выводятся также, как это сделано в bash в формате "command: {file_name}: Error"
Все ошибки логируются в `shell.log` с временными метками.
